其次就是，我其实还没有理解这个代码在干什么喵，但是我在这里学到了一些东西喵，一个就是，二分搜索的时候，可以初始化一个i= 0，这个可以用来表示在当前节点左边的子树大小喵，还有就是喵，二分搜索树的中序遍历是有序的喵，所以喵，有在k左上子树的节点个数就是k - 1喵

$$
k = 1 + \text{Size}(k_{\text{left}}) + \sum_{A \in \text{RightAncestors}} (\text{Size}(A_{\text{left}}) + 1)
$$

k代表：结点它自己喵 $\text{Size}(k_{\text{left}})$ 代表：它自己左子树里的所有小跟班喵。 $\sum$ 这一坨代表：头顶上所有往右拐的祖先，以及这些祖先的左子树里的所有小跟班喵。

void solve()

	int n,k;

	int i = 0;
	int l = 1,r = n;
	int x = 1;
	int cnt = 1;
	while(cnt <= n)

		int mid = (l + r)/2;
		if (k < mid)
		{
			r = mid - 1;
			i = 2 * i;
			x *= 2;
		}
		else if(k > mid)
		{
			l = mid + 1;
			i = 2 * i + 1;
			x = x * 2 + 1;
		}
		else
		{
			cout<<x<<' ';
			return;
		}
		cnt = cnt * 2 + 1;
	}
	cout<< cnt / 2 + k - i<<' ';
	return;
}//
signed main()
