[A Simple RBS Problem](https://codeforces.com/contest/2210/problem/D)

题目允许你交换任意两个不相交的合法括号子串。把这个操作放到树的视角来看，简直不要太直观：这其实就是把树上的两棵子树（或者子森林）连根拔起，然后互相交换它们生长的位置（嫁接）喵！

1. 叶子节点的数量绝对不变喵！ 也就是最小的不可分割的 () 的数量（即连续的左括号加右括号）。因为每次交换的都是合法的括号序列，而合法括号序列两端的字符绝对不可能因为拼接而凭空凑出新的 ()，也不会破坏原有的 ()。所以，s 和 t 中的 () 数量必须完全相等！
2. 最外层不可拆分的“保护壳”数量绝对不变喵！ 如果一个字符串整个被包裹在一对大括号里，比如 ( ... )，这意味着除了它自己，外面没有任何多余的合法括号可以和它内部的括号进行交换！所以，字符串外层有多少个连续的“全局嵌套外壳”（即整个字符串形如 (((...))) 的层数），这个层数是无论如何都无法打破的。s 和 t 的这个外壳层数必须完全一致！

## 至于为什么喵？

1. 为什么 () 的数量不变？
在树的结构里，里面没有任何内容的 ()，就是没有任何子节点的叶子节点喵！
你把一棵子树从这根树枝砍下来，嫁接到另一根树枝上，这棵子树上长着的叶子数量会凭空变多或者变少吗？当然不会呀笨蛋！所以无论你怎么像无头苍蝇一样瞎交换，整棵树的叶子节点总数绝对是不变的喵！

2. 为什么“全局外壳”不变？
如果一个括号序列长成 ((( ... ))) 这样，这意味着在树的视角下，这棵树的根节点只有唯一一个子节点，这个子节点也只有唯一一个子节点……这就形成了一根光秃秃的“单链主干”喵！
你想想看，交换操作要求你选出两个不相交的子树进行交换。在这根光秃秃、没有分叉的单链主干上，你根本不可能同时找出两个不相交的非空部分！所以，你只能在主干下面那些有分叉的茂密枝叶里进行交换，树顶上这几层单传的“主干”是无论如何也碰不到、拆不散的喵！这就对应了字符串外层无法打破的连续大括号！

所以，代码就会很好写了喵

```cpp
void solve()
{	
	int n;
	cin>>n;
	string s,t;
	cin>>s>>t;
	int ct901 = 0,ct902 = 0,ctw1 = 0,ctw2 = 0;
	stack<int> stk;
	vector<int> mt1(n),mt2(n);
	for (int i = 0 ; i < n ; i++)
	{
		if (s[i] == '(')
		{
			stk.emplace(i);
		}
		else
		{
			if (stk.top() == i - 1)
			{
				ct901++;
			}
			mt1[i] = stk.top();
			mt1[stk.top()] = mt1[i];
			stk.pop();
		}
	}
	for (int i = 0 ; i < n ; i++)
	{
		if (t[i] == '(')
		{
			stk.emplace(i);
		}
		else
		{
			if (stk.top() == i - 1)
			{
				ct902++;
			}
			mt2[i] = stk.top();
			mt2[stk.top()] = mt2[i];
			stk.pop();
		}
	}
	for (int i = 0 ; i < n ; i++)
	{
		if (mt1[n - i - 1] == i)
		{
			ctw1++;
		}
		else break;
	}
	for (int i = 0 ; i < n ; i++)
	{
		if (mt2[n - i - 1] == i)
		{
			ctw2++;
		}
		else break;
	}
	if (ct901 == ct902 && ctw1 == ctw2) cout<<"Yes\n";
	else cout<<"No\n";
	return;
}
```
