对于这道题喵，我vp的时候其实记得结论的喵，**假设换根为rt的(u,v)的lca就是相较于原来的根的 $lca(u,x)$ $lca(v,x)$ $lca(u,v)$ 这三个取根最深的节点喵**。

### 证明喵

设 $x = \text{LCA}(u, v)$（这是它们在原树的 LCA）。

**情况 A： $rt$ 在 $u$ 的分支里（但不在 $x$ 的上方）**

这时候你去拎 $rt$，$u$ 会跟着上去，它去跟 $v$ 汇合的交点就变成了 $\text{LCA}(u, rt)$ 喵。

此时你会发现 $\text{LCA}(v, rt)$ 还是等于 $x$，而 $\text{LCA}(u, rt)$ 显然比 $x$ 更深！

所以最深的是 $\text{LCA}(u, rt)$，结论成立喵。

**情况 B： $rt$ 在 $v$ 的分支里喵。**

同理可得，交点变成了 $\text{LCA}(v, rt)$，它比 $x$ 深，结论依然成立喵！

**情况 C： $rt$ 在 $x$ 的上方，或者在完全不相干的旁支里喵。**

这个时候你不管怎么拉扯 $rt$，$u$ 和 $v$ 想要相遇，还是只能苦哈哈地先走到 $x$，然后再一起朝 $rt$ 走喵！

这时候 $\text{LCA}(u, rt)$ 和 $\text{LCA}(v, rt)$ 要么等于 $x$，要么比 $x$ 还浅。所以三个里面最深的还是原来的 $\text{LCA}(u, v)$，结论再次完美成立喵！

## 现在你已经会换根lca了，接下来学习区间lca吧

还有就是对于普通的区间lca，我们当然可以火箭毛毛虫用线段树，或者st表来求喵。~~虽然在这道题目里面完全可以~~但是，这里要了解的前提完全不是用两个数据结构写的喵，**整个区间的lca是这个区间中dfn序最小的和dfn序最大的两个点的lca喵**

### 证明喵

令 $u \in S$ 为集合中 DFS 序最小的节点，即对于任意 $x \in S$，都有 $\text{dfn}(u) \le \text{dfn}(x)$ 喵。

令 $v \in S$ 为集合中 DFS 序最大的节点，即对于任意 $x \in S$，都有 $\text{dfn}(x) \le \text{dfn}(v)$ 喵。

令 $L = \text{LCA}(S)$，即整个集合 $S$ 的真正最近公共祖先喵。令 $M = \text{LCA}(u, v)$，也就是那两个极端节点的最近公共祖先喵。

因为 $L$ 是整个集合 $S$ 的公共祖先，而 $u$ 和 $v$ 都在集合 $S$ 里面，所以 $L$ 必然也是 $u$ 和 $v$ 的公共祖先喵。

根据 LCA（最近公共祖先）的定义，任何一个普通的公共祖先，必然是“最近”公共祖先的祖先（或者就是它本身）。

所以，$L$ 一定在 $M$ 的上方，我们记作 $L \preceq M$（ $\preceq$ 表示是其祖先或等于）喵。

既然 $L \preceq M$，那我们只要证明 $L$ 绝对不可能严格比 $M$ 高（即不可能 $L \prec M$），就能得出 $L = M$ 的唯一结论了喵！

假设本喵今天吃错了猫薄荷，非要认为 $L \prec M$（即 $L$ 是 $M$ 的严格祖先，比 $M$ 更靠近树根），会发生什么荒谬的事情呢喵？

既然 $L = \text{LCA}(S)$，但 $M$ 却在 $L$ 的下面，说明 $M$ 的势力范围不够大， $M$ 不能罩住 $S$ 里的所有节点 喵！

也就是说，在集合 $S$ 中，必定存在至少一个“叛逆”的节点 $w \in S$，使得 $w$ 不在以 $M$ 为根的子树中喵。

此时，我们来看看这个叛逆的 $w$ 和 $M$ 的相对位置关系，无非只有三种情况，每一种都会狠狠打主人的脸喵：

**情况 A：在 DFS 遍历 $M$ 的子树之前，就访问了 $w$ 所在的分支喵。**

如果是这样，那么 $w$ 的访问时间一定比 $M$ 子树里的所有节点都早。因为 $u$ 在 $M$ 的子树里，所以必定有 $\text{dfn}(w) < \text{dfn}(u)$ 喵！

打脸：这与“$u$ 是集合 $S$ 中 DFS 序最小的节点”的前提直接矛盾了喵！

**情况 B：在 DFS 遍历 $M$ 的子树之后，才访问了 $w$ 所在的分支喵。**

如果是这样，那么 $w$ 的访问时间一定比 $M$ 子树里的所有节点都晚。因为 $v$ 也在 $M$ 的子树里，所以必定有 $\text{dfn}(w) > \text{dfn}(v)$ 喵！

打脸：这又与“ $v$ 是集合 $S$ 中 DFS 序最大的节点”的前提矛盾了喵！

**情况 C： $w$ 恰好是 $M$ 的严格祖先（但在 $L$ 的下方）喵。**

如果 $w$ 挂在 $M$ 的头顶上，那 DFS 遍历时肯定是先路过 $w$，再深入到底下去遍历 $M$ 及其子树。这意味着 $\text{dfn}(w) < \text{dfn}(M) \le \text{dfn}(u)$ 喵。

打脸：同样得到了 $\text{dfn}(w) < \text{dfn}(u)$，再次与“ $u$ 是最小值”的前提矛盾喵！

你看，不论这个叛逆的节点 $w$ 躲在树的哪个角落，只要它不在 $M$ 的子树里，就会导致 $u$ 不是最小的，或者 $v$ 不是最大的！既然所有的退路都被数学逻辑堵死了，说明我们最初的假设“ $L \prec M$ ”是极其荒谬的错觉喵！

所以，真相只有一个：既然 $L \preceq M$ 且不能 $L \prec M$，那必定有 $L = M$ 喵！ 证明完毕！

## 现在你已经学会了区间lca和换根lca，接下来试试区间换根lca吧

那么这时候和普通的换根lca，区间lca有什么不同喵？

基于换根lca的结论喵，我们肯定是要找这个区间里和 $rt$ 这个换过来的根，在原来的树上最深的lca喵。虽然有些不严谨，但是我们明显可以注意到。我们第一个证明出来的结论，似乎是可以在区间上做出一定的推广喵

我们考虑区间的所有点，那么换根过来的lca一定是原来树上区间中和 $rt$ 最深的lca和原来树上的区间lca最深的那一个喵，严谨的证明在下面喵

### 证明喵

#### 引理 1：新 LCA 等于原 LCA 与所有点到新根 LCA 中的最深者喵【目标】

设原树为 $T_1$，以 $rt$ 为根的新树为 $T_{rt}$。

集合为 $S$ , $U = \text{LCA}_1(S)$ 。

我们要证明：

$$\text{LCA}_{rt}(S) = \text{argmax}_{v \in {U} cup {\text{LCA}_1(x, rt) \mid x \in S}} \text{depth}_1(v)$$

【证明】在新树 $T_{rt}$ 中，集合 $S$ 的 LCA 的本质是什么喵？它其实就是从 $rt$ 出发，通向所有 $x \in S$ 的路径的“最长公共前缀”的终点喵。

我们回到原树 $T_1$ 的视角。

对于任意一个 $x \in S$ , 从 $rt$ 到 $x$ 的路径是怎么走的？

必然是先从 $rt$ 向上爬到 $\text{LCA}_1(rt, x)$ ,然后再向下走到 $x$ 喵。

令 $D$ 为所有向上爬的顶点中，深度最深（离 $rt$ 最近）的那一个喵： 

$$D = \text{argmax}_{v \in {text{LCA}_1(x, rt) \mid x in S}} \text{depth}_1(v)$$

这意味着，从 $rt$ 出发通向所有 $x \in S$ 的路径，必然都会先重合地向上爬，至少爬到 $D$ 这个节点！在到达 $D$ 之前，任何人都不可能拐弯（因为都在唯一的祖先链上）喵。

那么到了 $D$ 之后，路径还会不会继续重合呢喵？分两种情况：

**情况 A： $D$ 严格在 $U$ 的上方( $\text{depth}_1(D) < \text{depth}_1(U)$ )**

说明所有节点在 $D$ 处还没有分开，它们必须继续携手向下走到原树的集结点 $U$ ,然后才在 $U$ 的各个子树中彻底散开。

此时公共前缀的终点是 $U$ ,所以 $\text{LCA}_{rt}(S) = U$ 喵。

**情况 B： $D$  在 $U$ 的下方或重合( $\text{depth}_1(D) \ge \text{depth}_1(U)$ )**

说明必然存在某个节点 $x_0 \in S$ ,它的路径在 $D$ 这个点就已经不再往上爬，而是拐弯向下进入了 $D$ 的某个旁支(或者 $x_0$ 刚好就是 $D$)

但同时，集合 $S$ 里肯定还有其他节点需要继续往上爬去寻找 $U$ 因此，路径在 $D$ 当场决裂！

公共前缀的终点就是 $D$，所以 $\text{LCA}_{rt}(S) = D$ 喵。

**【引理 1 结论】 综合 A 和 B，新 LCA 必然是 $U$ 和 $D$ 中深度更大的那一个，引理 1 证毕 $\blacksquare$ 喵！**

#### 引理 2：最大深度 $D$ 必然由 $Z_1$ 或 $Z_2$ 取得喵【目标】

我们要证明，根本不需要遍历集合 $S$ 里的所有点去求 $\text{LCA}_1(rt, x)$ 

那个最深的 $D$，必定在 $Z_1$ 或 $Z_2$ 处产生喵！

$$D = \max{\text{depth}_1(\text{LCA}_1(Z_1, rt)), \text{depth}_1(\text{LCA}_1(Z_2, rt)) }$$

【证明】还是盯住刚才那个 $D$。既然 $D$ 是集合 $S$ 中与 $rt$ 的最深 LCA，我们来看 $D$ 的子树喵。

$D$ 有一个包含了 $rt$ 的直接子树，我们记为 $Sub_{rt}$ 喵。

关键反证法： $Sub_{rt}$ 里面绝对不可能存在任何 $S$ 中的节点喵！

如果有某个 $y \in S$ 躲在 $Sub_{rt}$ 里，那 $y$ 和 $rt$ 的 LCA 至少也是 $Sub_{rt}$ 的根节点，其深度严格大于 $D$，这与“$D$ 是最大深度”的前提直接矛盾喵！

既然 $Sub_{rt}$ 是一片没有 $S$ 存在的“真空区”，而在 DFS 遍历中，一棵子树的 $dfn$ 必然是连续区间，设为 $[L_{sub}, R_{sub}]$ 喵。

显然， $dfn[rt]$ 被死死地包在这个真空区间里($L_{sub} \le dfn[rt] \le R_{sub}$)喵。

集合 $S$ 里的所有节点，它们的 $dfn$ 必须避开这个真空区，所以要么全部 $< L_{sub}$，要么全部 $> R_{sub}$ 喵。

回顾 $Z_1$ 和 $Z_2$ 的定义： $Z_1$  是 $S$ 中 $\le dfn[rt]$ 的最大者，所以它一定紧紧卡在 $L_{sub}$ 的左边界外喵！

$Z_2$ 是 $S$ 中 $\ge dfn[rt]$ 的最小者，所以它一定紧紧卡在 $R_{sub}$ 的右边界外喵！

我们已知产生 $D$ 的那个节点 $x_0 \in S$(即 $\text{LCA}_1(rt, x_0) = D$ )，它必须在 $D$ 的整棵大子树范围内，但又不在 $Sub_{rt}$ 内。

这意味着  $dfn[x_0]$ 虽然在真空区 $[L_{sub}, R_{sub}]$ 外面，但仍然被包在 $D$ 的大区间 $[L_D, R_D]$ 里面喵！

夹逼定理发威喵：既然 $x_0$ 这种离 $rt$ 比较远的节点，都能落在 $D$ 的势力范围 $[L_D, R_D]$ 内；

那么作为在 $dfn$ 序列上比 $x_0$ 更靠近真空区边界的 $Z_1$ 或 $Z_2$，至少有一个必定也会被“夹”在  $D$  的势力范围 $[L_D, R_D]$ 内喵！

只要 $Z_1$ 或 $Z_2$ 落在了 $D$ 的子树内，且不在 $Sub_{rt}$ 内，它和 $rt$ 的原树 LCA 就必定刚好是 $D$ 喵！

**【引理 2 结论】 $Z_1$ 和 $Z_2$ 完美捕获了最大深度 $D$，引理 2 证毕 $\blacksquare$ 喵**

int qpre(int k,int u,int v,int l,int r)
{
	if (info[v].v.val - info[u].v.val == 0) return 0;
	ll mid = l + (r - l) / 2;
	if (l == r) return l;
	if (mid >= k)
	{
		return qpre(k,info[u].ls,info[v].ls,l,mid);
	}
	int res = qpre(k,info[u].rs,info[v].rs,mid + 1,r);//我们每次都贪心地往右边找，如果找不到再回来
	if (res != 0) return res;
	else return qpre(k,info[u].ls,info[v].ls,l,mid);
}
int qnxt(int k,int u,int v,int l,int r)
{
	if (info[v].v.val - info[u].v.val == 0) return 0;
	if (l == r) return l;
	ll mid = l + (r - l) / 2;
	if (mid < k)
	{
		return qnxt(k,info[u].rs,info[v].rs,mid + 1,r);
	}
	int res = qnxt(k,info[u].ls,info[v].ls,l,mid);
	if (res != 0) return res;
	else return qnxt(k,info[u].rs,info[v].rs,mid + 1,r);
}
